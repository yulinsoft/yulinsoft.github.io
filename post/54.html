1. Secrets 未正确配置

3. Secrets 名称拼写错误

检查 secrets 的名称是否拼写正确，名称区分大小写。例如，MY_SECRET 和 my_secret 是不同的。

4. 权限问题

如果你在 Fork 仓库中运行工作流，默认情况下 secrets 不会传递到 Fork 仓库的 Actions 中。需要在工作流触发时手动启用 secrets。

5. Secrets 不会暴露给 PR

如果工作流是由外部 Pull Request 触发的，GitHub 出于安全考虑不会将 secrets 暴露给工作流。
